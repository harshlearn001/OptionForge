# Code of Conduct

## Our Pledge

We are committed to creating a welcoming, respectful and collaborative environment.

## Expected Behavior

- Be respectful.
- Be constructive.
- Welcome different perspectives.
- Focus on technical discussions.
- Help others learn.

## Unacceptable Behavior

- Harassment
- Personal attacks
- Discrimination
- Trolling
- Spam

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

Thank you for helping build a positive community around OptionForge.
